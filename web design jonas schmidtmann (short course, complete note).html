#14. design -> conversion

1. free gift: build trust
2. call to caption button
	strong, repeat it
	w/o spamming
	eg. in up-right navigation, center
3. grab user attention
	eg. pop-up
	convert well: do work.
4. tell user benefit
	on the call-to-action
5. don't ask more info than you need
	user don't want too many form
	use form with fewer fields
6. use social proof
	testimonials (often in box)
7. use urgency
	eg. now hurry immedient, 
	eg. limited time remaning deal
8. use scarcity
	when less, tend to value more.
	eg. only 1 in stock
